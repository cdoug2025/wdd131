My Chess Overview

Domain Name

The domain name I would use is my-chess-overview.org.
This website is my short beginners overview for chess, so this felt like a good name.

Purpose

The purpose is to give a short description of chess on the home page, a tutorial on a second page, and include a way to play a game on the third page.

Scenarios

Questions this site seeks to answer are:
What is chess?
How to play?

Colors

I chose blue as a primary color because it can be a good color for informational websites. I added a secondary orangish color for accents because it's around the opposite of blue.
The blue will be used in the header backround, a background color for any headings I want accented, and a font color for most headings to seperate them from the rest of the text.
The orange will be used to make things stand out from the blue when needed, like in the header.

Typography

I plan on using roboto. It's a good font and has been my default for the course. I don't think I will need any more fonts, so this will be used everywhere.
